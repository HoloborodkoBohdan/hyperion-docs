## About This Project

This is an **Odoo-related solution** that extends Odoo's core functionality with Ukrainian-specific features for FOP management. It integrates seamlessly with Odoo's HR, Accounting, and Payment modules while adding specialized capabilities for Ukrainian legal and accounting requirements.

## Key Features

### Employee Management
- FOP employee registration and profile management
- KVED (Ukrainian activity classification) code validation
- Tax group validation (3rd group)
- Responsible Account Manager assignment
- Integration with Tax Inspection Registry

### Contract Management
- Contract creation with automatic PDF generation
- Unique contract numbering (YYYYMMNNN format)
- Overlap prevention (one active contract per employee)
- Two PDF versions: with and without company facsimile
- Contract states: draft → running → archived

### Billing & Payments
- Automatic bill creation from batch payments
- Single "Total" product line approach
- Bill validation and posting to Odoo
- Payment reconciliation
- Integration with Odoo accounting module

### Reporting
- PDF contract generation via wkhtmltopdf
- Queue-based report generation
- File storage with audit logging
- Cloud storage integration

### Tax & Compliance
- Tax Inspection Registry synchronization
- KVED code compliance checking
- Ukrainian tax requirements validation
- Audit trail for all operations

## System Integration

The UA FOP Management System integrates with:

- **Odoo ERP** - Core business platform for HR, Accounting, and Payments
- **Tax Inspection Registry** - Ukrainian tax authority data synchronization
- **PostgreSQL** - Primary database for all application data
- **Report Engine (wkhtmltopdf)** - PDF generation
- **Cloud Storage** - File and attachment storage
